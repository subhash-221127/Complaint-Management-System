How Automatic Escalation Works

When a complaint is not resolved within 5 minutes by an L1 officer, it automatically escalates to their assigned L2 supervisor. If the L2 officer also doesn't resolve it within 5 more minutes, it escalates to the L3 Director. Set up supervisor assignments below so escalation always reaches the right person. If no supervisor is set, the system will distribute to the least-loaded officer of the next level.
